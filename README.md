# 🌿 PixelEcology
> *A Darwinian nightmare in your browser — watch life evolve, hunt, and die in real-time.*

---

## Description

PixelEcology is a living ecosystem simulation where hundreds of autonomous pixel-creatures evolve, hunt, reproduce, and die right inside your browser. Each species carries a unique genetic color signature that mutates across generations, producing mesmerizing emergent patterns you simply can't stop watching. Think Conway's Game of Life crossed with a lava lamp and a Darwinian fever dream.

---

## ✨ Features

- 🧬 **Genetic color mutation** — every generation subtly shifts, creating lineages you can visually trace
- 🦎 **Autonomous creature behavior** — each pixel-creature moves, reacts, and makes decisions independently
- 🌊 **Emergent ecosystem dynamics** — hunting, reproducing, and dying produce complex patterns from simple rules
- 📐 **Fullscreen, responsive canvas** — fills any screen size and adapts on resize without breaking the simulation
- ⚡ **Real-time animation** — smooth, continuous simulation loop powered by the native Canvas API
- 🖤 **Zero-bloat frontend** — no frameworks, no bundlers, just raw JavaScript and a canvas element

## 🚀 Installation & Running

Make sure you have [Node.js](https://nodejs.org/) installed, then follow these steps:

```bash
# 1. Clone the repository
git clone https://github.com/sami-23/PixelEcology.git

# 2. Navigate into the project directory
cd PixelEcology

# 3. Install dependencies
npm install

# 4. Start the server
npm start
```

Then open your browser and visit:

```
http://localhost:3000
```

## 🛠️ Tech Stack

| Layer | Technology | Purpose |
|---|---|---|
| **Runtime** | Node.js | Server-side JavaScript environment |
| **Server** | Express.js `^4.18.2` | Serves static files and handles routing |
| **Rendering** | HTML5 Canvas API | Real-time pixel drawing and animation loop |
| **Frontend** | Vanilla JavaScript | Creature logic, simulation engine, event handling |
| **Markup** | HTML5 / CSS3 | Fullscreen canvas layout, zero-margin styling |
